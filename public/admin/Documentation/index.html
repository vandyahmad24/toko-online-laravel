Overview

Meter is a fully responsive admin dashboard template built with Bootstrap Framework.There are a lot of useful components, well organized and structured properly. It is easy to customize and coded in developer friendly.It can be used for all type of web applications like project management system, CMS, CRM for personal and business purposes.

  • Full functional chat
  • Chartjs
  • Morris Chart
  • Flot Charts
  • Multi Select
  • Timepicker
  • Form Validation
  • Google maps
  • Vector maps
  • Range Slider
  • Dynamic Table
  • Well structured code
  • Form Component
  • Fully Responsive pages
  • Multi File upload
  • Validation Form
  • Bootstrap 3x
  • Easy to customize
  • Flat UI with clean style
  • Font Icons
